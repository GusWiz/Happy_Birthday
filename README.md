# Happy Birthday Site - Yuji Itadori

A celebratory birthday website dedicated to Yuji Itadori from the anime **Jujutsu Kaisen**.

## About

This project is a simple, interactive happy birthday site built with HTML and CSS. It features a personalized birthday message along with multiple gift sections that reveal animated GIFs when you hover over them. Each gift represents a different sentiment or characteristic about the birthday person.

## Purpose

This site was created as a practice project to strengthen skills in:
- HTML structure and semantics
- CSS styling and layout (Flexbox)
- Interactive hover effects
- Working with background images and animations

## Features

- Centered, festive design with gradient background
- Circular profile image of the birthday person
- Multiple "gift" sections with hover-activated GIF animations
- Responsive text styling with custom fonts
- Footer with attribution to image sources

## Live Site

🎉 [Visit the deployed site on Netlify](https://happybirthday-yuji.netlify.app/) 

## Project Structure

- `index.html` - Main HTML structure
- `style.css` - All styling and hover effects
- `images/` - Directory containing the profile image and GIF files
